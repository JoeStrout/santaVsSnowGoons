# Santa vs. the Snow Goons

This is the classic "Daleks" game, but with a Christmas theme! Made for [Mini Micro](https://miniscript.org/MiniMicro]).

Clone this repo, mount the user.minidisk folder in Mini Micro, and run startup.ms to play.

Or, [play online at itch.io](https://joestrout.itch.io/snow-goons).
